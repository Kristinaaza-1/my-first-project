# Research.md — рынок AI-ботов/агентов/автоматизаций, сделанных соло-разработчиками (2026)

Контекст: перед сборкой AI-помощника для косметологов (см. [спеку](thoughts/shared/specs/2026-07-15-ai-inbox-assistant-mvp.md) и [no-code план](thoughts/shared/specs/2026-07-15-ai-inbox-assistant-nocode-plan.md)) мы уже выяснили, что прямая ниша («AI-ассистент + запись через YCLIENTS/Altegio для бьюти-мастеров») занята зрелыми игроками — Бьюти Бот, Wahelp, Suvvy, AI Beauty Bot. Чтобы понять, как в принципе устроен рынок соло-построенных AI-продуктов вокруг, и найти реальные углы дифференциации/монетизации, пять параллельных агентов исследовали пять смежных категорий: Telegram/WhatsApp-боты, AI-ассистенты, автономные AI-агенты, no-code автоматизации (n8n/Make), и продукты-«упакованные инструкции для агентов».

Все продукты ниже — реальные, найдены через поиск (Product Hunt, Reddit, Indie Hackers, vc.ru, YouTube, GitHub), большинство запущено за последние 6-12 месяцев соло-разработчиками или маленькими командами.

## Категория 1: Telegram/WhatsApp-боты

### 1. Cobura — Telegram-канал на автопилоте
Бот курирует и подписывает видео про космос для канала, весь код написан через Claude Code без единой строчки вручную.
- **Путь**: бот сканирует источники → генерирует подпись → присылает владельцу на утверждение → владелец одобряет за 30 сек → публикует; Яндекс вставляет рекламные посты.
- **Монетизация**: доля от рекламной сети Яндекса, ~10 000 ₽/мес при 12 500 подписчиках, минус ~2 000 ₽ расходов.
- **Сложность**: 🟢 просто.
- **Через Claude Code**: разворачивает бота через BotFather, пишет логику подбора контента, планировщик, деплой на VPS — всё по скриншотам и словесным инструкциям владельца.
- **Стек**: Telegram Bot API, Python, LLM API, Hetzner VPS.
- **Для ниши Кристины**: паттерн «бот делает черновик → человек утверждает одним действием» — прямо переиспользуется как схема эскалации в её MVP.

### 2. Vyneron — захват задач через Telegram
Личный органайзер: пишешь боту текст/фото/голосовое — оно превращается в структурированную задачу.
- **Монетизация**: freemium, BYOK на AI-провайдера, 0% наценки сверху.
- **Сложность**: 🟡 средне (мультиязычный парсинг дат, синхронизация с веб-приложением).
- **Для ниши Кристины**: важный урок — единый общий бот с привязкой аккаунта работает лучше, чем требование от каждого пользователя заводить свой токен (потеря ~20% регистраций у Vyneron).

### 3. ExpenseBot — учёт расходов через Telegram
Голосовое/текстовое сообщение → строка в Google Sheets.
- **Монетизация**: DIY бесплатно (Make.com) или «сделаем под ключ» за $149 разово, готово за 24 часа.
- **Сложность**: 🟢 просто.
- **Для ниши Кристины**: модель «разовая оплата за настройку» реалистичнее продавать первым мастерам, чем сразу подписку.

### 4. invioTrack — инвойсы для фрилансеров через Telegram
- **Монетизация**: freemium — Free / Pro $20/мес / Business $40/мес; Stripe только для подписки, не для приёма платежей клиента (чтобы не становиться платёжным оператором).
- **Сложность**: 🟡 средне.
- **Для ниши Кристины**: принцип «не становиться платёжным оператором» — важно: бот не должен принимать оплату сам, только направлять на существующие способы оплаты мастера.

### 5. text.ai — AI-ассистент через SMS/WhatsApp/Telegram
Без установки приложения, работает в трёх каналах на одном бэкенде.
- **Монетизация**: пока бесплатно (YC-стартап), 600k+ сообщений за 3 месяца без платного маркетинга.
- **Для ниши Кристины**: подтверждает, что мультиканальность (Telegram+WhatsApp) реализуема на одном бэкенде с самого начала.

### 6. AvBot — ближайший структурный аналог
Агрегирует сообщения с Avito в единый Telegram/MAX-инбокс, опционально с AI-автоответом.
- **Монетизация**: тарифы 799₽/1 499₽/2 499₽ в месяц по возможностям AI, + White-Label +5 000₽/мес, AI считается отдельно по расходу.
- **Сложность**: 🔴 сложно (мультиаккаунт, интеграции с amoCRM/Bitrix24).
- **Для ниши Кристины**: практически та же бизнес-форма (агрегация сообщений → эскалация не-рутины → тарифы по AI-возможностям), но для перекупщиков на Avito, а не для бьюти-мастеров — подтверждает, что модель переносима между нишами.

### 7. Кластер «клик-в-WhatsApp» инструментов (WASender, ChatWith, SuperLemon)
Готовые ссылки/виджеты «написать нам в WhatsApp» для сайтов/Instagram.
- **Монетизация**: подписка, ~$29-32k/мес выручки у каждого (SuperLemon — 65 000+ Shopify-магазинов).
- **Сложность**: 🟢 просто.
- **Для ниши Кристины**: почти бесплатный «быстрый выигрыш» — кнопка «Записаться через WhatsApp» с готовым текстом в шапку Instagram мастера, ещё до готовности полного AI-ассистента.

### 8. Голосовой транскрибатор для WhatsApp
Пересылаешь голосовое — получаешь текст за секунды.
- **Для ниши Кристины**: клиенты часто шлют голосовые вопросы — встроить транскрипцию в приёмный конвейер (голос → текст → классификация → маршрутизация) — дёшево и решает реальное трение.

**Источники**: [Vyneron](https://www.indiehackers.com/post/three-weeks-of-shipping-vyneron-solo-redesigned-app-central-telegram-bot-natural-language-dates-in-6-languages-bc70e6611f) · [ExpenseBot](https://www.indiehackers.com/post/expensebot-track-business-expenses-via-telegram-0020c6897f) · [invioTrack](https://www.indiehackers.com/post/launched-inviotrack-a-telegram-invoicing-bot-for-freelancers-no-app-no-card-fees-3a33ac9c8e) · [text.ai](https://www.producthunt.com/products/text-ai) · [AvBot](https://avbot.ru/landing/) · [WhatsApp $30k/mo](https://www.indiehackers.com/post/make-over-30k-mo-using-whatsapp-fj39ttTpmPCBemFo6D6g) · [Claude Code Telegram-бот кейс](https://androidinsider.ru/polezno-znat/kak-claude-code-sdelal-mne-telegram-bota-on-vedyot-za-menya-kanal-i-zarabatyvaet-10k-rublej-v-mesyacz.html)

## Категория 2: AI-ассистенты / чат-боты

### 1. irel — «AI-секретарь внутри WhatsApp»
Настраиваешь повторяющиеся «Jobs» (разбор почты, поздравления, утренний дайджест) — работает в фоне.
- **Монетизация**: $18/мес (годовая), $30/мес (помесячно), $349 навсегда.
- **Для ниши Кристины**: концепция «Jobs» (фоновые повторяющиеся задачи, а не только реактивный чат) — перенести на напоминания о повторной записи, отправку инструкций подготовки, поздравления с днём рождения.

### 2. Scarlyfy — WhatsApp AI-бот записи для independent-докторов в Латинской Америке
Почти прямой шаблон продукта Кристины, только для врачей.
- **Монетизация**: от $22/мес, 14 дней триал.
- **Реальный опыт основателя**: 4-6 недель ожидания Meta Business Verification для «около-медицинских» ботов — одобрение идёт быстрее, если кейс промаркирован как «запись на приём», а не «AI-ассистент». Его цитата: «вы конкурируете не с софтом, а с привычками» (мастера всё ещё ведут заметки вручную).
- **Для ниши Кристины**: изучить внимательно — то же самое узкое место с Meta, та же психология сопротивления привычке.

### 3. Telegram AI-менеджер + автосоздание лида в CRM (кейс n8n, «Гурьев PRO AI»)
Линейный сценарий: Telegram → AI отвечает по базе знаний → собирает контакт → создаёт лид в CRM (включая YCLIENTS).
- **Сложность**: 🟢 просто — линейный workflow в n8n, без кастомного бэкенда.
- **Для ниши Кристины**: простейший первый строительный блок, который стоит собрать в первую очередь — он напрямую тренирует именно интеграцию с YCLIENTS.

### 4. HumanInbox — AI-ассистент почты через Telegram/WhatsApp
- **Монетизация**: плоская подписка $99/мес.
- **Для ниши Кристины**: принцип «никакого нового приложения, просто пишешь своему ассистенту там, где уже общаешься» — мастер не установит ничего нового, если ассистент не живёт в её уже привычном Telegram/WhatsApp.

### 5. Solvea — AI-ресепшн «отвечает, записывает и продаёт»
Готовые отраслевые шаблоны, включая **med-spa** — почти прямой аналог бьюти-салона.
- **Монетизация**: freemium, 11 000 бесплатных кредитов/мес.
- **Позиционируется** как «первый AI customer service продукт, построенный через vibe coding» — прямое подтверждение подхода Кристины.
- **Для ниши Кристины**: идея готового «шаблона по отрасли» (пред-заполненные вопрос-ответы) — мастеру нужно только подставить свои цены/адрес, не создавать логику с нуля.

### 6-7. Dialzara, Slang.ai — голосовые AI-ресепшионисты
Голосовые агенты для малого бизнеса/ресторанов; Slang.ai буквально родился из истории «тётя-основателя держала салон красоты и не справлялась с потоком звонков».
- **Сложность**: 🔴 сложно (телефония, голосовой пайплайн) — не стартовая точка для no-code новичка.
- **Для ниши Кристины**: голос — это правдоподобная «фаза 2» после того, как чат-версия подтверждена, не стартовая точка.

### 8. Chatbase — конструктор RAG-чат-ботов, $9M ARR соло-фаундера без инвесторов
Загружаешь документы → бот отвечает по ним, с эскалацией при неуверенности.
- **Монетизация**: free 50 сообщений/мес, далее $32/$120/$400/мес.
- **Для ниши Кристины**: reusable «ядро» — механика "загрузи прайс/инструкции → получи бота, отвечающего по ним" — именно то, что должно быть внутри её ассистента (база: прайс, адрес, инструкции подготовки).

**Источники**: [irel](https://www.indiehackers.com/post/i-built-an-ai-secretary-that-lives-in-your-whatsapp-and-its-not-just-another-chatbot-d9b1bdc8f6) · [Scarlyfy](https://www.indiehackers.com/post/i-built-a-whatsapp-ai-bot-for-doctors-in-peru-launched-3-weeks-ago-0-paying-customers-and-stuck-waiting-for-meta-to-approve-my-app-ffc73befd1) · [Telegram AI-менеджер](https://vc.ru/ai/2718142-kak-ii-menedzher-v-telegram-uvelichivaet-vyruchku-biznesa-na-30) · [HumanInbox](https://www.producthunt.com/products/humaninbox) · [Solvea](https://www.producthunt.com/products/solvea) · [Dialzara](https://dialzara.com/pricing) · [Slang.ai](https://www.slang.ai/pricing) · [Chatbase](https://solofounders.com/blog/9m-arr-zero-investors-yasser-elsaid-on-bootstrapping-chatbase-as-a-solo-founder)

## Категория 3: Автономные AI-агенты

### 1. Pancake — «сотрудник в Slack», ведущий компанию из одного человека
Ролевые агенты (GTM, продукт, операции) с общей «памятью компании».
- **Монетизация**: $49/мес + токен-пакеты $50-1000.
- **Для ниши Кристины**: модель «один агент, одна задача, всегда на связи» напрямую переносится — вместо Slack интерфейс Telegram/WhatsApp, задача агента — «вести переписку с клиентами + записывать» — это буквально её MVP.

### 2. Wobo — агент, ищущий и подающий заявки на работу автономно
- **Для ниши Кристины**: механика «автономного заполнения форм в чужих системах» — резервный способ бронирования в YCLIENTS/Altegio через имитацию браузера, если API окажется недоступен.

### 3. AllyHub — браузерный агент, «запоминающий» структуру сайтов
- **Для ниши Кристины**: похожий резервный вариант — агент один раз «изучает» интерфейс конкретного личного кабинета YCLIENTS и переиспользует паттерн для всех будущих записей.

### 4. CoreNews — автономный новостной портал одного русскоязычного разработчика
Четыре агента (SEO, редактор, SMM, DevOps) ведут сайт целиком без редакции, координируются через мастер-агента «Гермес».
- **Для ниши Кристины**: прямой архитектурный шаблон — цепочка узкоспециализированных агентов + оркестратор: один классифицирует сообщение, один готовит ответ, один проверяет календарь, один эскалирует чувствительные темы.

### 5. Zaturn/Androva — команда маркетинговых AI-агентов для малого бизнеса
Черновик → одобрение человеком → публикация.
- **Для ниши Кристины**: паттерн «черновик + подтверждение человеком» как способ снять тревогу мастера перед полной автономностью на старте — можно предложить пилотный период именно в таком режиме.

### 6. Голосовые AI-агенты записи на базе Retell/Vapi
- **Для ниши Кристины**: ближайший голосовой аналог — та же связка «календарь + эскалация», просто голосом вместо чата; следующий канал после того, как чат подтверждён.

### 7. Соло-операторы, монетизирующие мониторинг соцсетей (Reddit/Instagram DM lead-mining)
Паттерн «мониторинг → классификация → черновик → отправка» — зеркальная версия механики Кристины (только исходящая, не входящая).
- **Для ниши Кристины**: подтверждает, что связка «классифицировать → маршрутизировать» — проверенный, реализуемый в одиночку паттерн.

## Категория 4: No-code автоматизации (n8n/Make/Zapier)

### 1. Inrō — автоматизация Instagram DM + AI-продажи
- **Монетизация**: freemium, Pro €12.99/мес, Managed Account от €200/мес (услуга «сделаем за вас»).
- **Для ниши Кристины**: та же механика (комментарий/DM → квалификация → запись) работает для Instagram мастера — часто это её основной канал привлечения, может питать тот же бэкенд, что и WhatsApp-ассистент.

### 2. N8N Starter Kit (Евгений Карташов, RU-рынок)
Набор из 8 готовых workflow (включая «AI-секретарь в Telegram»), продаётся разово за 7 900 ₽.
- **Для ниши Кристины**: прямой шаблон того, как её собственный workflow можно позже упаковать и перепродать другим мастерам как отдельный «стартовый набор» ~$50-100 разово, параллельно с подпиской.

### 3. Пакеты n8n-шаблонов на Gumroad/Whop
Соло-разработчики продают JSON своих workflow как цифровой товар, $19-299 за штуку; один автор зарабатывает ~$3 200/мес пассивно на 5 шаблонах.
- **Для ниши Кристины**: низкорисковый побочный источник дохода — облегчённая версия её ассистента как Gumroad-товар для мастеров, которые хотят собрать сами.

### 4. WhatsApp AI Bot Template (Whapi.Cloud + n8n, open-source)
Готовый бесплатный шаблон именно под связку WhatsApp + n8n + OpenAI.
- **Важный выбор**: официальный WhatsApp Cloud API (безопаснее, больше настройки) vs Whapi.Cloud (быстрее, риск бана) — решение, с которым Кристина тоже столкнётся напрямую.
- **Для ниши Кристины**: структурно ближайший аналог её стека — стоит изучить логику маршрутизации команд (цифровое меню + текстовый AI-fallback) как дешёвый способ не дать AI напутать с двусмысленным запросом.

### 5. Модель «AI-агентство на n8n» — retainer вместо продажи софта
Соло-операторы строят кастомные workflow под конкретного клиента и берут абонентскую плату $2000-5000/мес за клиента.
- **Для ниши Кристины**: запасной/параллельный путь, если самообслуживаемый продукт медленно набирает обороты — можно предложить нескольким мастерам «настрою всё за вас» как платную услугу более высокого чека, используя их фидбек для доработки массового продукта.

### 6. Бесплатный шаблон как воронка в платный курс (Nate Herk)
- **Для ниши Кристины**: механика «бесплатный полезный мини-инструмент как воронка» может сработать и у неё — например, бесплатный «генератор автоответов» или шаблон FAQ для мастеров, который потом ведёт к платному AI-ассистенту.

### 7. Community-шаблон генерации лидов (Apify + GPT + WhatsApp-ссылки)
- **Для ниши Кристины**: приём с готовой `wa.me`-ссылкой с предзаполненным текстом напрямую переиспользуем — записаться через WhatsApp без установки чего-либо ещё.

## Категория 5: Упакованные «инструкции для агентов» как продукт

### 1. God of Prompt — библиотека промптов, разовая продажа
От $37 до $150 за полный бандл, 100 000+ пользователей.
- **Для ниши Кристины**: прямой шаблон низкопорогового продукта — «Набор скриптов/промптов для косметологов» (шаблоны ответов клиентам, апселл, тексты после процедуры) как разовая продажа тем, кто ещё не готов к полному AI-ассистенту — входная ступень в её экосистему.

### 2. AIPRM — расширение-браузер + маркетплейс промптов
Подписка от $20 до $999+/мес.

## Сквозные выводы для проекта Кристины

**Переиспользуемое технологическое «ядро», которое встречается почти в каждом продукте:**
1. Мессенджер как единственный интерфейс (Telegram/WhatsApp) — без установки отдельного приложения.
2. LLM с function calling как «мозг», классифицирующий запрос (рутина / эскалация).
3. Календарь/CRM API (или имитация браузера как резервный вариант, если API не даёт нужного).
4. Правило эскалации к человеку — встречается буквально во всех надёжных продуктах, не только в спеке Кристины.
5. RAG-слой поверх прайса/инструкций мастера («ядро» Chatbase) как база знаний для рутинных ответов.

Ни один успешный продукт не требовал сложных агентных фреймворков (LangChain/AutoGPT) — везде 2-4 связанных API, что подтверждает реалистичность подхода «no-code + Claude Code» без технического кофаундера.

**Реальные риски, подтверждённые независимо несколькими источниками:**
- Задержка одобрения Meta Business Verification для WhatsApp/Instagram — 4-6 недель в кейсе Scarlyfy, с советом маркировать кейс как «запись на приём», а не «AI-ассистент», для более быстрого одобрения.
- «Вы конкурируете не с софтом, а с привычками» (Scarlyfy) — мастера, которые годами вели заметки вручную, не переключатся только из-за наличия технологии.
- Не становиться платёжным оператором самим (invioTrack) — направлять оплату через существующие способы мастера, а не обрабатывать деньги внутри бота.

**Дополнительные источники дохода/дистрибуции, которые стоит держать в уме помимо основной подписки:**
- Разовая оплата за настройку ($149-2000) как более лёгкая первая продажа, чем подписка.
- Продажа облегчённого no-code шаблона отдельным мастерам как побочный продукт (Gumroad/Whop, $30-300).
- Бесплатный мини-инструмент (набор скриптов/FAQ-шаблон) как воронка в основной продукт — хорошо ложится на её образовательную экосистему.
- Ценообразование за диалог как альтернатива фиксированной подписке — не выбрано сейчас, но встречается у нескольких игроков (Poe, AvBot).

**Ценовые ориентиры на рынке** (для калибровки будущей подписки): однoканальные чат/запись-боты — $18-30/мес; более полнофункциональные с несколькими фичами — $99+/мес; голосовые — от $300/мес и выше.
